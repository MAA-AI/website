# MaaAI 官网

MaaAI 官方网站源码，基于 Next.js 构建。

## 技术栈

- Next.js 15
- TypeScript
- Tailwind CSS

## 功能

- 响应式设计
- 亮色/暗色主题切换（支持跟随系统）
- 中英文切换（自动检测系统语言）

## 开发

```bash
pnpm install
pnpm run dev
```

访问 http://localhost:3000

## 构建

```bash
pnpm run build
```
